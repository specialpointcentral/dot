# Agent 环境安装指引

当用户要求进行环境安装、配置恢复或 dotfiles 部署时，按以下规则执行。

## 执行来源

**`README.md` 是标准安装指南。** 阅读 README 中的步骤并逐步执行。

## 步骤标记语义

README 中的步骤隐含以下分类，执行时注意区分：

| 标记 | 含义 | Agent 行为 |
|------|------|-----------|
| `[自动]` | 可直接执行的命令 | 直接运行 |
| `[手动]` | 需要人工交互（输入密码、编辑密钥等） | 提示用户手动完成，不要代替执行 |
| `[检查]` | 验证步骤是否成功 | 执行并确认输出符合预期 |

## 密钥安全

- **绝不** 在输出中暴露密钥内容（API key、token 等）
- 密钥文件（`~/.secrets.zsh`、`~/.codex/auth.json`）由用户手动编辑，Agent 只负责从模板复制并提示用户填写
- 确认密钥文件权限为 `600`

## 执行流程

1. 阅读 `README.md` 中的「快速开始」和 Step 0–6
2. 按顺序执行符号链接创建、模板复制等 `[自动]` 步骤
3. 遇到 `[手动]` 步骤时提示用户操作
4. 如需安装工具（zsh、tmux、node 等），参考 README「工具安装指南」章节
5. 完成后运行验证命令确认配置正确
